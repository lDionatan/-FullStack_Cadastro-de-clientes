# 📋 FullStack Cadastro de Clientes

Sistema fullstack para cadastro e gerenciamento de clientes, desenvolvido com **React no frontend** e **Node.js no backend**.

---

## ⚠️ Aviso Importante

> 🚨 Este é um **projeto antigo** desenvolvido utilizando uma versão anterior do **Prisma ORM com JavaScript**, cuja configuração **não é mais utilizada atualmente**.

* O Prisma moderno é baseado em **TypeScript**
* A estrutura deste projeto pode não refletir as práticas atuais
* Pode ser necessário atualizar dependências para execução completa

👉 Este repositório é mantido como **registro de aprendizado e evolução**

---

## 🚀 Tecnologias Utilizadas

### Frontend

* React
* JavaScript
* CSS

### Backend

* Node.js
* Express
* Prisma ORM (**versão antiga - JavaScript**)

---

## 📁 Estrutura do Projeto

```
/Cadastro-Clientes
  /frontend   → Aplicação React
  /backend    → API Node.js
```

---

## ⚙️ Como Executar o Projeto

### 🔹 1. Clonar o repositório

```
git clone https://github.com/lDionatan/-FullStack_Cadastro-de-clientes.git
```

---

### 🔹 2. Rodar o Backend

```
cd backend
npm install
npm run dev
```

---

### 🔹 3. Rodar o Frontend

```
cd frontend
npm install
npm start
```

---

## 🧠 Objetivo do Projeto

Este projeto foi desenvolvido com foco em:

* Praticar integração entre frontend e backend
* Criar um CRUD completo de clientes
* Entender fluxo de requisições HTTP
* Trabalhar com banco de dados via ORM

---

## ⚠️ Limitações

* Uso de versão antiga do Prisma
* Configuração desatualizada em relação ao padrão atual
* Pode exigir ajustes para rodar em ambientes modernos
